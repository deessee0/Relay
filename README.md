# Relay

Relay is a local-first AI copilot for incident reporting, triage, and team handoffs in unreliable-connectivity environments.

## Hackathon targets
- Amazon Nova AI Hackathon
- PowerSync AI Hackathon

## Product idea
Relay helps teams capture incidents, maintain a synchronized timeline, generate AI summaries, estimate severity, and prepare next-step handoffs even when connectivity is unreliable.

## MVP
- Create incident reports
- Add structured notes and timeline events
- Generate AI summary
- Estimate severity with rationale
- Suggest next actions
- Prepare clean demo-first architecture for dual submission

## Planned stack
- TypeScript
- Next.js
- Node.js
- Amazon Nova
- PostgreSQL
- PowerSync + Sync Streams
- SQLite/local-first data model

Initial scaffold in progress.
