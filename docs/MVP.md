# Relay MVP Freeze

## Must-have
- Persistent incident object model
- Incident creation flow
- Timeline/note model
- AI summary output
- Severity recommendation with rationale
- Next-step recommendation
- Role-specific handoff view
- Demo script and architecture writeup

## Nice-to-have
- Browser-based interface on top of the same product flow
- Better role-based output formatting
- Small amount of polish that improves demo clarity

## Not now
- PowerSync-specific infrastructure
- Voice features
- Image understanding
- Multi-agent orchestration
- Complex auth
- Production infra polish

## Product rule
If a feature does not make the Nova demo more convincing, it is not part of the MVP.
